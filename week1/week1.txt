Instal rustup which is a installation/version manager for rust
- curl https://sh.rustup.rs -sSf | sh
- relog into the shell for the env to be updated
- cargo and rustc are now available in the PATH

My first Rust Program
- Hello world tutorial: https://rustbyexample.com/hello.html
- compile: rustc hello_world.rs
- run: ./hello_world
- it worked :D
